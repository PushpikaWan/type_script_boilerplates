# Inital steps 
1. compile code - 
```
tsc index.ts
```
2. then run generated js file - 
``` 
node index.js
```
or we can install ts-node to combine those command 
then we can run 

```
ts-node index.ts
```

#initalize npm

```
npm init -y
````

#  Tools

tools to helps us run typescript code on browser

npm install -g parcel-bundler
